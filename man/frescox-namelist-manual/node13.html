Input SFRESCOX format

This is an program which search for a minimum when comparing the outputs of FRESCOX with sets of data, using the MINUIT search routines.

The inputs for SFRESCOX specify the FRESCOX input and output files, the number and types of search variables, and the experimental data sets.
