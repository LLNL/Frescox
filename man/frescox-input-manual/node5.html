Partitions and Excited States

Card 6

(for each partition IC=1,2,... repeat until a blank card 8) : 
A8,      2F8.4,           I4,   L1,1X, A8,      2F8.4,           F8.4,
NAME(1,),MASS(1,),ZZ(1,), NEX(),PWF(), NAME(2,),MASS(2,),ZZ(2,), QVAL(IC),

name-----mass-----charge-              name-----mass----charge-  Qvalue   
----projectile----------               ---------target--------

I8,        F8.4, 2I4
READSTATES,PRMAX,LPMAX1(IC),MIXPOT

Abs(NEX) = number of pairs of excitations of projectile and/or target,

PWF = T, if acceleration of long-range Coulomb couplings to be performed when card 1a present. Default is T.

QVAL = value of this partition, relative to the -values of the other partitions, for Ep = Et = 0.

If NEX 0, then cross sections for this partition are not printed.

If PRMAX 0, then use PRMAX for matching radius in this partition

If LPMAX10, then LPMAX=LPMAX1-1 is the maximum partial wave in this partition. Applied for incident energies ELAB below ELPMAX.

MIXPOT governs the couplings that are generated (in this partition) between levels that have different KP potential indices. It does not affect any couplings when the KP index values are identical. 8ex1
0: no couplings (default)
= 1: couplings only to or from the first state in that partition
= 2: couplings only according to the KP value of the destination level
= 3: all couplings produced by either KP value.

Card 7

(repeat NEX times before the next Card 6) : 
F4.1,I2,   I2,    F8.4,2F4.1, I4,1X,A1
Jp,  COPYp,BANDp ,Ep,  KKp,Tp,KP,PWF,
-------projectile state-----

F4.1,I2,   I2,    F8.4,2F4.1,  2L2,         2I4
Jt,  COPYt,BANDt ,Et,  KKt,Tt, EXCH,IGNORE, INFAM,OUTFAM
-----target state-----------

0ex This card is repeated NEX times, IA=1,NEX

7ex1 For both projectile (p) and target (t), the following are read :

7ex1 J = spin of the state

7ex1 BAND 0 for positive parity state, and 0 for negative parity. The value of BAND defines the rotational band : all states of the same BAND may be coupled by rotational mechanisms.

7ex1 If COPY is positive, then this level is a copy of a previous level number (COPY IA) in the same partition.

7ex1 If COPYp 0, then this level is an exchange copy of the same IA numbered level in the previous partition abs(COPY). That is, a copy with projectile and target nuclei exchanged.

7ex1 If COPYt = -IC, then the target state is an exchanged projectile. If that state has spin zero, then only even partial waves are allowed.

7ex1 COPYs are needed to define identical states of one nucleus in different pairings IA. Spectroscopic amplitudes (see Cards 17) should be defined for only the original `copied' level.

7ex1 E = excitation energy of state relative to ground state.

7ex1 KK = K of rotation band containing this state. (If zero, use spin of first level in this band.)

7ex1 T = isospin of this state (if zero, use INT((mass-2*charge)/2)) (The T value is not used by this version of FRESCO).

7ex1 KP = index of optical potential for this pair of excited states.

7ex1 PWF= T to use CRCWFN coupled Coulomb wave functions in this partition (see Cards 1,1a)
If zero, use as default KP = IC, the number of this partition.

10ex1 If EXCH = T calculate and write out all cross sections and amplitudes for .

10ex1 If IGNORE = T, ignore the convergence of this state pair (see IPS discussion on Card 4).

10ex1 If INFAM 0, read in complex amplitudes later to add to calculated scattering amplitudes (see cards A3). (If INFAM 0, read in only one amplitude (non spin-flip) for each scattering angle.)

10ex1 If OUTFAM 0, write out complex amplitudes later to add to calculated scattering amplitudes (see cards A3). (If OUTFAM 0, write out only one amplitude (non spin-flip) for each scattering angle: the average of diagonal amplitudes.)

Card A3

10ex1 If INFAM on a Card 7 is non-zero, then read on file INFAM, on a new card for each scattering angle, some complex amplitudes , to be added to those calculated by the program before the cross sections are calculated.

10ex1 If INFAM 0, then only one f(external) is read, and this is used for all the diagonal f(mM:mM), the off-diagonal values being zero.

10ex1 If INFAM 0, then all the are read in.
