Glossary of Indexing Variables

10ex1 IC,IC1,IC2 index to mass partitions

10ex1 IA,IB index to projectile-target pairs of excited states within each partition

10ex1 KP index to table of potentials, for scattering, binding, and folding

10ex1 KN index to single-particle bound states

10ex1 KN1,KN2 two-particle bound states are given for a range (RMIN to RNN) of internucleon separations, and are stored in the MSP table from KN1 to KN2 inclusive.

10ex1 CP index to the coupling set up
