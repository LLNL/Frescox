Mass and charge constants

To exactly reproduce previous FRESCO runs using = 0.0481960, you will need to enter to this version the numbers: unitmass=1.007335d0; finec=137.5648d0. To exactly reproduce using = 0.0478326, you need input unitmass=0.999740d0; finec=137.0455d0, but only the default input of unitmass=1d0; finec=137.03599d0 is completely accurate.
